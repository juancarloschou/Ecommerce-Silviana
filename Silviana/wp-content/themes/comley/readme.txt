Description
-------------------

Comley is a Simple, Clean and Responsive WordPress Theme which automatically adapts to the screen size, ensuring that your content is always displayed beautifully no matter what device visitors are using. Theme comes with a lot of features like multiple page templates, multiple blog layouts and theme options using the built-in WP Customizer. 

Comley also supports WP Customizer, Custom Menu, Custom Background and Custom Header, enabling flexible configuration.

Setting Up Front Page
--------------------------

By default, your front page looks like a blog. However, you can make it look like the screenshot by following these steps.
1. Go to Dashboard => Appearance => Customize
2. Click on 'Static Front Page'
3. Select 'A static page' radio option
4. Select a static page for 'Front Page' and another page for 'Posts Page'
5. Now go to Edit 'front page'
6. Select 'Home Page' from Template
7. In order to upload banner image, go to Dashboard => Appearance => Customize => Themes Settings => Banner Image Change (You can upload any size for the banner image but recommended size is 1920 x 460 pixels)
