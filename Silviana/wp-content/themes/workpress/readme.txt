WorkPress is an ideal WordPress theme for those who wish to create an impressive web presence. WorkPress is easy to use, provides everything you need to create a great looking website. It is professional, smooth and sleek, with a clean modern layout, for almost any business types: agency, freelance, blog, startup, portfolio, corporate, firms, law firms, digital media agency, architecture firms, real estate firms, food, fashion etc. The theme is developed using Latest Bootstrap CSS framework that makes it friendly for all the modern devices like mobiles, tablets, laptops etc.

The Parent theme Rambo needs to be installed for this WorkPress to work.
Install
-------
1. Download "Rambo" parent theme from https://wordpress.org/themes/rambo/
2. Go to your WordPress dashboard and select Appearance > Install, click Upload link.
3. Upload both Rambo and WorkPress Theme, then activate the WorkPress Theme
